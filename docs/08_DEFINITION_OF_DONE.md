\## System DoD

A system is "done" when:

\- It has a clear owner service/class

\- It has acceptance criteria tested in Play Mode

\- It has minimal debug visibility (logs or HUD)

\- It is documented in the relevant docs file



---



\## Task DoD

A task is "done" when:

\- All subtasks completed

\- Acceptance criteria satisfied

\- Changes committed to GitHub

\- Any new scripts placed in correct folders

\- Naming conventions followed



---



\## Milestone DoD

A milestone is "done" when:

\- Required tasks in roadmap are complete

\- Prototype is playable end-to-end

\- No critical console errors

\- A short gameplay recording can be produced (optional)
